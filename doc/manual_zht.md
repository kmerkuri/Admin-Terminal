# ssh調用助手使用
註冊ssh調用助手以便於在用戶windows電腦上調用putty軟件連接至服務器,使用方法如下
將軟件目錄helper下webterminal ssh helper.exe安裝包文件安裝至需要連接至服務器的客戶端上。
在服務器列表頁面點擊putty按鈕即可連接至需要連接的服務器
